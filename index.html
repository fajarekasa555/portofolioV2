🙋 About Me

Sedikit tentang saya

Hai! Saya Rizki Pratama, seorang Full-Stack Developer dari Jawa Timur yang gemar membangun aplikasi web yang tidak hanya fungsional, tapi juga menyenangkan untuk digunakan.

Perjalanan coding saya dimulai dari rasa ingin tahu bagaimana sebuah website bisa "hidup" — dan sejak saat itu, saya tidak pernah berhenti eksplorasi. Saya menikmati proses dari whiteboard sampai deployment, dari desain antarmuka hingga optimasi query database.

Di luar layar laptop, saya juga seorang digital artist — menggambar karakter dan ilustrasi adalah cara saya mengisi ulang energi kreatif. Dua dunia ini ternyata saling melengkapi: kemampuan visual membuat saya lebih peka terhadap UI/UX, dan logika programming membuat karya seni saya lebih terstruktur.

Saat ini saya membuka diri untuk peluang freelance, kolaborasi open source, maupun posisi full-time. Mari ngobrol! ☕
