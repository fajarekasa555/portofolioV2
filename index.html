About

A bit about me

Hai! Saya Fajar Eka Sandiyuda, Full Stack Web Developer dari Malang, Jawa Timur. Saya memiliki pengalaman lebih dari 3 tahun membangun aplikasi web enterprise — mulai dari sistem pemerintahan, manajemen rumah sakit, hingga e-commerce dan CMS.

Perjalanan saya dimulai di SMKN 1 Boyolangu jurusan Rekayasa Perangkat Lunak, lalu berlanjut ke Politeknik Negeri Malang dengan IPK 3.95/4.00. Saya juga aktif berkontribusi di dunia profesional melalui magang di Sekawan Media dan saat ini bekerja sebagai Full Stack Developer di Haltec.

Di luar coding, saya adalah seorang digital illustrator dan penggemar bahasa — saya berhasil meraih sertifikasi JLPT N5 dan JFT Basic (setara N4), bahkan memenangkan Kompetisi Kanji antar SMK se-Jawa Timur. Dua dunia ini saling melengkapi: kreativitas seni memperhalus kepekaan UI/UX saya, dan logika programming membuat desain lebih sistematis.

Saat ini terbuka untuk peluang Full-Time, Remote, atau Freelance. Mari ngobrol!
